Template Daftar Aset dan Inventaris Kantor Excel Gratis disiapkan untuk membantu pekerjaan rutin tanpa perlu membangun format dari nol. Ganti data contoh di sheet **Daftar Aset**, lalu gunakan **Ringkasan** untuk membaca kondisi data secara cepat.

## Fitur Utama

- Sheet **Cara Pakai** berisi urutan mulai yang singkat.
- Sheet **Daftar Aset** memakai tabel dengan filter agar data mudah dicari dan diurutkan.
- Sheet **Ringkasan** menghitung jumlah data dan nilai aset tercatat secara otomatis.
- Status memiliki pilihan yang konsisten agar pekerjaan tindak lanjut lebih mudah.
- File tidak memakai macro dan dapat disesuaikan sesuai kebutuhan.

## Struktur Workbook

**Cara Pakai** menjelaskan langkah awal. **Daftar Aset** adalah tempat mengisi data utama. **Ringkasan** merangkum jumlah data, status, dan metrik penting dari sheet input.

## Cara Pakai

1. Beri setiap aset kode unik agar mudah dilacak.
2. Catat lokasi, jumlah, nilai perolehan, dan kondisi terakhir.
3. Gunakan kondisi untuk menandai aset yang perlu diperiksa.
4. Perbarui daftar saat ada pembelian, perpindahan, atau penghapusan aset.

## Contoh Alur Penggunaan

File sudah berisi data ilustrasi agar susunan kolom mudah dipahami. Hapus atau timpa contoh tersebut sebelum memakai file untuk kebutuhan sendiri.

## Rumus dan Logika

Ringkasan memakai `COUNTA` untuk menghitung data yang sudah diisi dan `COUNTIF` untuk membaca status. Karena itu, isi satu item per baris dan pilih status secara konsisten.

## Tips Modifikasi

Tambahkan data melalui baris berikutnya di sheet **Daftar Aset** agar filter tabel tetap nyaman dipakai. Bila istilah status diubah, gunakan penulisan yang sama pada setiap baris supaya rekap tidak terpecah. Simpan satu salinan file setiap periode bila ingin menyimpan arsip.

## Batasan Template

Template ini tidak menghitung penyusutan aset dan bukan pengganti sistem manajemen aset atau pembukuan formal.

## Pertanyaan yang Sering Ditanyakan (FAQ)

**Apakah contoh data bisa dihapus?**

Bisa. Data awal hanya ilustrasi; ganti dengan data sendiri sebelum file dipakai rutin.

**Apakah file ini memakai macro?**

Tidak. File menggunakan format `.xlsx` tanpa macro.

**Apakah bisa dipakai di Google Sheets?**

Bisa diimpor ke Google Sheets. Tampilan grafik atau validasi pilihan mungkin memerlukan penyesuaian kecil karena mesin spreadsheet berbeda.
