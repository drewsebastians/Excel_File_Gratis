Template Surat Jalan Excel Gratis disiapkan untuk membantu pekerjaan rutin tanpa perlu membangun format dari nol. Ganti data contoh di sheet **Surat Jalan**, lalu gunakan **Ringkasan** untuk membaca kondisi data secara cepat.

## Fitur Utama

- Sheet **Cara Pakai** berisi urutan mulai yang singkat.
- Sheet **Surat Jalan** memakai tabel dengan filter agar data mudah dicari dan diurutkan.
- Sheet **Ringkasan** menghitung jumlah data dan jumlah barang dikirim secara otomatis.
- Status memiliki pilihan yang konsisten agar pekerjaan tindak lanjut lebih mudah.
- File tidak memakai macro dan dapat disesuaikan sesuai kebutuhan.

## Struktur Workbook

**Cara Pakai** menjelaskan langkah awal. **Surat Jalan** adalah tempat mengisi data utama. **Ringkasan** merangkum jumlah data, status, dan metrik penting dari sheet input.

## Cara Pakai

1. Isi tanggal dan nomor surat jalan untuk setiap pengiriman.
2. Tulis penerima, barang, jumlah, dan satuannya.
3. Pilih status pengiriman agar dokumen yang belum selesai mudah dicari.
4. Cetak atau salin baris yang diperlukan sebagai dasar surat jalan operasional.

## Contoh Alur Penggunaan

File sudah berisi data ilustrasi agar susunan kolom mudah dipahami. Hapus atau timpa contoh tersebut sebelum memakai file untuk kebutuhan sendiri.

| Tanggal | No. Surat Jalan | Penerima | Barang | Jumlah |

## Rumus dan Logika

Ringkasan memakai `COUNTA` untuk menghitung data yang sudah diisi dan `COUNTIF` untuk membaca status. Karena itu, isi satu item per baris dan pilih status secara konsisten.

## Tips Modifikasi

Tambahkan data melalui baris berikutnya di sheet **Surat Jalan** agar filter tabel tetap nyaman dipakai. Bila istilah status diubah, gunakan penulisan yang sama pada setiap baris supaya rekap tidak terpecah. Simpan satu salinan file setiap periode bila ingin menyimpan arsip.

## Batasan Template

Template ini adalah catatan pengiriman sederhana dan bukan dokumen legal atau bukti pajak.

## Pertanyaan yang Sering Ditanyakan (FAQ)

**Apakah contoh data bisa dihapus?**

Bisa. Data awal hanya ilustrasi; ganti dengan data sendiri sebelum file dipakai rutin.

**Apakah file ini memakai macro?**

Tidak. File menggunakan format `.xlsx` tanpa macro.

**Apakah bisa dipakai di Google Sheets?**

Bisa diimpor ke Google Sheets. Tampilan grafik atau validasi pilihan mungkin memerlukan penyesuaian kecil karena mesin spreadsheet berbeda.
