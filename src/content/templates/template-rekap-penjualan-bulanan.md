Template Rekap Penjualan Bulanan Excel Gratis membantu mencatat total penjualan dalam format yang rapi. File berisi petunjuk, sheet Rekap Bulanan, dan ringkasan otomatis dengan data ilustrasi yang bisa diganti. Isi data pada sheet kerja yang relevan, lalu gunakan ringkasan sebagai pembacaan dari data sumber, bukan angka yang diketik ulang.

## Mulai dengan Data Sumber

1. Buka sheet **Cara Pakai** untuk melihat urutan input yang tersedia.
2. Masukkan satu catatan per baris pada sheet kerja utama.
3. Gunakan pilihan atau format yang sudah disediakan agar ringkasan dapat membaca data secara konsisten.
4. Cocokkan hasil ringkasan dengan beberapa baris sumber sebelum menggunakannya.

## Fakta File dan Batasan

Workbook yang diperiksa memiliki 3 sheet, format xlsx, dan tidak memakai macro. Buka dan uji salinan file sebelum dipakai rutin, terutama bila menggunakan aplikasi spreadsheet selain Microsoft Excel.

Template Rekap Penjualan Bulanan Excel Gratis disiapkan untuk membantu pekerjaan rutin tanpa perlu membangun format dari nol. Ganti data contoh di sheet **Rekap Bulanan**, lalu gunakan **Ringkasan** untuk membaca kondisi data secara cepat.

## Fitur Utama

- Sheet **Cara Pakai** berisi urutan mulai yang singkat.
- Sheet **Rekap Bulanan** memakai tabel dengan filter agar data mudah dicari dan diurutkan.
- Sheet **Ringkasan** menghitung jumlah data dan total penjualan secara otomatis.
- Status memiliki pilihan yang konsisten agar pekerjaan tindak lanjut lebih mudah.
- File tidak memakai macro dan dapat disesuaikan sesuai kebutuhan.

## Struktur Workbook

**Cara Pakai** menjelaskan langkah awal. **Rekap Bulanan** adalah tempat mengisi data utama. **Ringkasan** merangkum jumlah data, status, dan metrik penting dari sheet input.

## Cara Pakai

1. Pilih bulan yang sedang direkap dengan format YYYY-MM.
2. Masukkan jumlah terjual dan harga rata-rata setiap produk atau jasa.
3. Kolom Total Penjualan menghitung otomatis.
4. Bandingkan hasil per produk sebelum membuat keputusan stok atau promosi.

## Contoh Alur Penggunaan

File sudah berisi data ilustrasi agar susunan kolom mudah dipahami. Hapus atau timpa contoh tersebut sebelum memakai file untuk kebutuhan sendiri.

| Bulan | Produk/Jasa | Kategori | Jumlah Terjual | Harga Rata-rata |

| 2026-07 | Kopi Susu | Minuman | 120 | 18.000 |

## Rumus dan Logika

Kolom Total Penjualan memakai rumus `=D4*E4` pada baris contoh. Saat menambah baris, salin rumusnya atau perluas Excel Table agar pola perhitungan tetap ikut.

## Tips Modifikasi

Tambahkan data melalui baris berikutnya di sheet **Rekap Bulanan** agar filter tabel tetap nyaman dipakai. Bila istilah status diubah, gunakan penulisan yang sama pada setiap baris supaya rekap tidak terpecah. Simpan satu salinan file setiap periode bila ingin menyimpan arsip.

## Pertanyaan yang Sering Ditanyakan (FAQ)

**Apakah contoh data bisa dihapus?**

Bisa. Data awal hanya ilustrasi; ganti dengan data sendiri sebelum file dipakai rutin.

**Apakah file ini memakai macro?**

Tidak. File menggunakan format `.xlsx` tanpa macro.

**Apakah bisa dipakai di Google Sheets?**

Bisa diimpor ke Google Sheets. Tampilan grafik atau validasi pilihan mungkin memerlukan penyesuaian kecil karena mesin spreadsheet berbeda.
