Template Daftar Piutang Pelanggan Excel Gratis disiapkan untuk membantu pekerjaan rutin tanpa perlu membangun format dari nol. Ganti data contoh di sheet **Daftar Piutang**, lalu gunakan **Ringkasan** untuk membaca kondisi data secara cepat.

## Fitur Utama

- Sheet **Cara Pakai** berisi urutan mulai yang singkat.
- Sheet **Daftar Piutang** memakai tabel dengan filter agar data mudah dicari dan diurutkan.
- Sheet **Ringkasan** menghitung jumlah data dan total sisa piutang secara otomatis.
- Status memiliki pilihan yang konsisten agar pekerjaan tindak lanjut lebih mudah.
- File tidak memakai macro dan dapat disesuaikan sesuai kebutuhan.

## Struktur Workbook

**Cara Pakai** menjelaskan langkah awal. **Daftar Piutang** adalah tempat mengisi data utama. **Ringkasan** merangkum jumlah data, status, dan metrik penting dari sheet input.

## Cara Pakai

1. Masukkan nilai invoice dan tanggal jatuh tempo setiap pelanggan.
2. Perbarui kolom Pembayaran ketika uang diterima.
3. Kolom Sisa Piutang menghitung selisih secara otomatis.
4. Filter status Belum Lunas untuk daftar tindak lanjut.

## Contoh Alur Penggunaan

File sudah berisi data ilustrasi agar susunan kolom mudah dipahami. Hapus atau timpa contoh tersebut sebelum memakai file untuk kebutuhan sendiri.

| Tanggal | No. Invoice | Pelanggan | Jatuh Tempo | Nilai Invoice |
| --- | --- | --- | --- | --- |
| 2026-07-01 | INV-101 | Toko Sinar Pagi | 2026-07-08 | 450.000 |
| 2026-07-03 | INV-102 | CV Maju Bersama | 2026-07-10 | 800.000 |
| 2026-07-05 | INV-103 | Kantin Sehat | 2026-07-12 | 320.000 |

## Rumus dan Logika

Kolom Sisa Piutang memakai rumus `=E4-F4` pada baris contoh. Saat menambah baris, salin rumusnya atau perluas Excel Table agar pola perhitungan tetap ikut.

## Tips Modifikasi

Tambahkan data melalui baris berikutnya di sheet **Daftar Piutang** agar filter tabel tetap nyaman dipakai. Bila istilah status diubah, gunakan penulisan yang sama pada setiap baris supaya rekap tidak terpecah. Simpan satu salinan file setiap periode bila ingin menyimpan arsip.

## Batasan Template

Status dan tanggal tindak lanjut perlu diperbarui manual; template tidak mengirim pengingat otomatis.

## Pertanyaan yang Sering Ditanyakan (FAQ)

**Apakah contoh data bisa dihapus?**

Bisa. Data awal hanya ilustrasi; ganti dengan data sendiri sebelum file dipakai rutin.

**Apakah file ini memakai macro?**

Tidak. File menggunakan format `.xlsx` tanpa macro.

**Apakah bisa dipakai di Google Sheets?**

Bisa diimpor ke Google Sheets. Tampilan grafik atau validasi pilihan mungkin memerlukan penyesuaian kecil karena mesin spreadsheet berbeda.
