Template Daftar Harga Produk dan Jasa Excel Gratis disiapkan untuk membantu pekerjaan rutin tanpa perlu membangun format dari nol. Ganti data contoh di sheet **Daftar Harga**, lalu gunakan **Ringkasan** untuk membaca kondisi data secara cepat.

## Fitur Utama

- Sheet **Cara Pakai** berisi urutan mulai yang singkat.
- Sheet **Daftar Harga** memakai tabel dengan filter agar data mudah dicari dan diurutkan.
- Sheet **Ringkasan** menghitung jumlah data dan nilai daftar harga secara otomatis.
- Status memiliki pilihan yang konsisten agar pekerjaan tindak lanjut lebih mudah.
- File tidak memakai macro dan dapat disesuaikan sesuai kebutuhan.

## Struktur Workbook

**Cara Pakai** menjelaskan langkah awal. **Daftar Harga** adalah tempat mengisi data utama. **Ringkasan** merangkum jumlah data, status, dan metrik penting dari sheet input.

## Cara Pakai

1. Ubah produk atau jasa contoh di sheet Daftar Harga.
2. Isi harga modal dan harga jual dalam angka biasa.
3. Pilih status Aktif atau Nonaktif untuk menandai item yang masih ditawarkan.
4. Gunakan Ringkasan Harga untuk memeriksa jumlah item dan total nilai harga jual contoh.

## Contoh Alur Penggunaan

File sudah berisi data ilustrasi agar susunan kolom mudah dipahami. Hapus atau timpa contoh tersebut sebelum memakai file untuk kebutuhan sendiri.

| Kode | Produk/Jasa | Kategori | Satuan | Harga Modal |
| --- | --- | --- | --- | --- |
| PRD-001 | Kopi Susu 1L | Minuman | Botol | 22.000 |
| PRD-002 | Snack Box | Makanan | Paket | 9.000 |
| JAS-001 | Desain Poster | Jasa | Paket | 75.000 |

## Rumus dan Logika

Ringkasan memakai `COUNTA` untuk menghitung data yang sudah diisi dan `COUNTIF` untuk membaca status. Karena itu, isi satu item per baris dan pilih status secara konsisten.

## Tips Modifikasi

Tambahkan data melalui baris berikutnya di sheet **Daftar Harga** agar filter tabel tetap nyaman dipakai. Bila istilah status diubah, gunakan penulisan yang sama pada setiap baris supaya rekap tidak terpecah. Simpan satu salinan file setiap periode bila ingin menyimpan arsip.

## Batasan Template

Daftar ini membantu menata informasi harga. Harga akhir, diskon, dan pajak tetap perlu disesuaikan dengan kebijakan usaha.

## Pertanyaan yang Sering Ditanyakan (FAQ)

**Apakah contoh data bisa dihapus?**

Bisa. Data awal hanya ilustrasi; ganti dengan data sendiri sebelum file dipakai rutin.

**Apakah file ini memakai macro?**

Tidak. File menggunakan format `.xlsx` tanpa macro.

**Apakah bisa dipakai di Google Sheets?**

Bisa diimpor ke Google Sheets. Tampilan grafik atau validasi pilihan mungkin memerlukan penyesuaian kecil karena mesin spreadsheet berbeda.
