Template Kas Kecil Excel Gratis membantu mencatat saldo kas kecil dalam format yang rapi. File berisi petunjuk, sheet Catatan Kas Kecil, dan ringkasan otomatis dengan data ilustrasi yang bisa diganti. Isi data pada sheet kerja yang relevan, lalu gunakan ringkasan sebagai pembacaan dari data sumber, bukan angka yang diketik ulang.

## Mulai dengan Data Sumber

1. Buka sheet **Cara Pakai** untuk melihat urutan input yang tersedia.
2. Masukkan satu catatan per baris pada sheet kerja utama.
3. Gunakan pilihan atau format yang sudah disediakan agar ringkasan dapat membaca data secara konsisten.
4. Cocokkan hasil ringkasan dengan beberapa baris sumber sebelum menggunakannya.

## Fakta File dan Batasan

Workbook yang diperiksa memiliki 3 sheet, format xlsx, dan tidak memakai macro. Buka dan uji salinan file sebelum dipakai rutin, terutama bila menggunakan aplikasi spreadsheet selain Microsoft Excel.

Template Kas Kecil Excel Gratis disiapkan untuk membantu pekerjaan rutin tanpa perlu membangun format dari nol. Ganti data contoh di sheet **Catatan Kas Kecil**, lalu gunakan **Ringkasan** untuk membaca kondisi data secara cepat.

## Fitur Utama

- Sheet **Cara Pakai** berisi urutan mulai yang singkat.
- Sheet **Catatan Kas Kecil** memakai tabel dengan filter agar data mudah dicari dan diurutkan.
- Sheet **Ringkasan** menghitung jumlah data dan saldo kas kecil secara otomatis.
- Status memiliki pilihan yang konsisten agar pekerjaan tindak lanjut lebih mudah.
- File tidak memakai macro dan dapat disesuaikan sesuai kebutuhan.

## Struktur Workbook

**Cara Pakai** menjelaskan langkah awal. **Catatan Kas Kecil** adalah tempat mengisi data utama. **Ringkasan** merangkum jumlah data, status, dan metrik penting dari sheet input.

## Cara Pakai

1. Isi dana awal atau pengisian kembali sebagai Jenis Isi.
2. Catat setiap pengeluaran kecil dan nomor bukti bila ada.
3. Gunakan status untuk menandai transaksi yang sudah diperiksa.
4. Cocokkan saldo ringkasan dengan uang fisik secara berkala.

## Contoh Alur Penggunaan

File sudah berisi data ilustrasi agar susunan kolom mudah dipahami. Hapus atau timpa contoh tersebut sebelum memakai file untuk kebutuhan sendiri.

## Rumus dan Logika

Ringkasan membedakan baris masuk atau pendapatan dari baris keluar atau beban dengan `SUMIFS`, kemudian menghitung nilai bersih dari selisih keduanya.

## Tips Modifikasi

Tambahkan data melalui baris berikutnya di sheet **Catatan Kas Kecil** agar filter tabel tetap nyaman dipakai. Bila istilah status diubah, gunakan penulisan yang sama pada setiap baris supaya rekap tidak terpecah. Simpan satu salinan file setiap periode bila ingin menyimpan arsip.

## Batasan Template

Saldo menjadi akurat hanya bila semua penerimaan dan pengeluaran kas kecil dicatat.

## Pertanyaan yang Sering Ditanyakan (FAQ)

**Apakah contoh data bisa dihapus?**

Bisa. Data awal hanya ilustrasi; ganti dengan data sendiri sebelum file dipakai rutin.

**Apakah file ini memakai macro?**

Tidak. File menggunakan format `.xlsx` tanpa macro.

**Apakah bisa dipakai di Google Sheets?**

Bisa diimpor ke Google Sheets. Tampilan grafik atau validasi pilihan mungkin memerlukan penyesuaian kecil karena mesin spreadsheet berbeda.
