Template Absensi Karyawan Sederhana Excel Gratis membantu mencatat jumlah catatan hadir dalam format yang rapi. File berisi petunjuk, sheet Absensi, dan ringkasan otomatis dengan data ilustrasi yang bisa diganti. Isi data pada sheet kerja yang relevan, lalu gunakan ringkasan sebagai pembacaan dari data sumber, bukan angka yang diketik ulang.

## Mulai dengan Data Sumber

1. Buka sheet **Cara Pakai** untuk melihat urutan input yang tersedia.
2. Masukkan satu catatan per baris pada sheet kerja utama.
3. Gunakan pilihan atau format yang sudah disediakan agar ringkasan dapat membaca data secara konsisten.
4. Cocokkan hasil ringkasan dengan beberapa baris sumber sebelum menggunakannya.

## Fakta File dan Batasan

Workbook yang diperiksa memiliki 3 sheet, format xlsx, dan tidak memakai macro. Buka dan uji salinan file sebelum dipakai rutin, terutama bila menggunakan aplikasi spreadsheet selain Microsoft Excel.

Template Absensi Karyawan Sederhana Excel Gratis disiapkan untuk membantu pekerjaan rutin tanpa perlu membangun format dari nol. Ganti data contoh di sheet **Absensi**, lalu gunakan **Ringkasan** untuk membaca kondisi data secara cepat.

## Fitur Utama

- Sheet **Cara Pakai** berisi urutan mulai yang singkat.
- Sheet **Absensi** memakai tabel dengan filter agar data mudah dicari dan diurutkan.
- Sheet **Ringkasan** menghitung jumlah data dan jumlah catatan hadir secara otomatis.
- Status memiliki pilihan yang konsisten agar pekerjaan tindak lanjut lebih mudah.
- File tidak memakai macro dan dapat disesuaikan sesuai kebutuhan.

## Struktur Workbook

**Cara Pakai** menjelaskan langkah awal. **Absensi** adalah tempat mengisi data utama. **Ringkasan** merangkum jumlah data, status, dan metrik penting dari sheet input.

## Cara Pakai

1. Isi satu baris untuk satu orang pada satu tanggal.
2. Pilih status kehadiran agar istilah selalu konsisten.
3. Tambahkan jam masuk dan keluar bila diperlukan.
4. Gunakan ringkasan sebagai hitungan dasar, lalu cek data sumber sebelum dipakai untuk keputusan formal.

## Rumus dan Logika

Ringkasan memakai `COUNTA` untuk menghitung data yang sudah diisi dan `COUNTIF` untuk membaca status. Karena itu, isi satu item per baris dan pilih status secara konsisten.

## Tips Modifikasi

Tambahkan data melalui baris berikutnya di sheet **Absensi** agar filter tabel tetap nyaman dipakai. Bila istilah status diubah, gunakan penulisan yang sama pada setiap baris supaya rekap tidak terpecah. Simpan satu salinan file setiap periode bila ingin menyimpan arsip.

## Batasan Template

Template tidak menghitung gaji, lembur, atau kepatuhan ketenagakerjaan. Terapkan aturan internal dan peraturan yang berlaku secara terpisah.

## Pertanyaan yang Sering Ditanyakan (FAQ)

**Apakah contoh data bisa dihapus?**

Bisa. Data awal hanya ilustrasi; ganti dengan data sendiri sebelum file dipakai rutin.

**Apakah file ini memakai macro?**

Tidak. File menggunakan format `.xlsx` tanpa macro.

**Apakah bisa dipakai di Google Sheets?**

Bisa diimpor ke Google Sheets. Tampilan grafik atau validasi pilihan mungkin memerlukan penyesuaian kecil karena mesin spreadsheet berbeda.
