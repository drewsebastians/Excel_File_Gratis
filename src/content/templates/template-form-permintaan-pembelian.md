Template Form Permintaan Pembelian Excel Gratis disiapkan untuk membantu pekerjaan rutin tanpa perlu membangun format dari nol. Ganti data contoh di sheet **Permintaan Pembelian**, lalu gunakan **Ringkasan** untuk membaca kondisi data secara cepat.

## Fitur Utama

- Sheet **Cara Pakai** berisi urutan mulai yang singkat.
- Sheet **Permintaan Pembelian** memakai tabel dengan filter agar data mudah dicari dan diurutkan.
- Sheet **Ringkasan** menghitung jumlah data dan jumlah permintaan secara otomatis.
- Status memiliki pilihan yang konsisten agar pekerjaan tindak lanjut lebih mudah.
- File tidak memakai macro dan dapat disesuaikan sesuai kebutuhan.

## Struktur Workbook

**Cara Pakai** menjelaskan langkah awal. **Permintaan Pembelian** adalah tempat mengisi data utama. **Ringkasan** merangkum jumlah data, status, dan metrik penting dari sheet input.

## Cara Pakai

1. Catat kebutuhan barang atau jasa dalam satu baris.
2. Isi jumlah dan estimasi biaya agar prioritas mudah dibandingkan.
3. Pilih status Menunggu, Disetujui, atau Ditolak.
4. Filter kolom Status saat ingin menindaklanjuti permintaan yang belum selesai.

## Contoh Alur Penggunaan

File sudah berisi data ilustrasi agar susunan kolom mudah dipahami. Hapus atau timpa contoh tersebut sebelum memakai file untuk kebutuhan sendiri.

| Tanggal | No. Permintaan | Peminta | Barang/Jasa | Jumlah |
| --- | --- | --- | --- | --- |
| 2026-07-12 | PR-001 | Dapur | Gula Aren | 15 |

| 2026-07-13 | PR-003 | Pemasaran | Cetak Banner | 1 |

## Rumus dan Logika

Ringkasan memakai `COUNTA` untuk menghitung data yang sudah diisi dan `COUNTIF` untuk membaca status. Karena itu, isi satu item per baris dan pilih status secara konsisten.

## Tips Modifikasi

Tambahkan data melalui baris berikutnya di sheet **Permintaan Pembelian** agar filter tabel tetap nyaman dipakai. Bila istilah status diubah, gunakan penulisan yang sama pada setiap baris supaya rekap tidak terpecah. Simpan satu salinan file setiap periode bila ingin menyimpan arsip.

## Batasan Template

Template ini mendukung pencatatan permintaan; alur persetujuan dan bukti pembelian perlu dikelola sesuai kebijakan masing-masing.

## Pertanyaan yang Sering Ditanyakan (FAQ)

**Apakah contoh data bisa dihapus?**

Bisa. Data awal hanya ilustrasi; ganti dengan data sendiri sebelum file dipakai rutin.

**Apakah file ini memakai macro?**

Tidak. File menggunakan format `.xlsx` tanpa macro.

**Apakah bisa dipakai di Google Sheets?**

Bisa diimpor ke Google Sheets. Tampilan grafik atau validasi pilihan mungkin memerlukan penyesuaian kecil karena mesin spreadsheet berbeda.
