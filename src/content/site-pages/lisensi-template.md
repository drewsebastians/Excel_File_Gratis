Halaman ini menjelaskan panduan lisensi penggunaan template ExcelGratis pada website, bukan nasihat hukum. Ketentuan khusus yang ditampilkan pada halaman template tertentu dapat melengkapi ketentuan umum ini.

## Penggunaan yang diizinkan

Anda boleh:

- menggunakan template untuk kebutuhan pribadi;
- menggunakan template untuk kebutuhan internal bisnis;
- memodifikasi template untuk kebutuhan Anda sendiri;
- mengisi template dengan data Anda sendiri;
- menghasilkan laporan internal;
- menggunakan hasil template dalam pekerjaan yang dilakukan untuk klien;
- memberikan file yang sudah selesai atau sudah diisi data kepada klien;
- memodifikasi template sebagai bagian dari penyelesaian pekerjaan klien, selama template kosong atau original tidak didistribusikan atau dijual sebagai produk template tersendiri.

## Penggunaan yang dilarang

Anda tidak boleh:

- menjual ulang template kosong;
- menjual ulang template original;
- mengunggah template ke website lain;
- menyediakan mirror unduhan langsung;
- mendistribusikan template sebagai bagian dari koleksi template lain;
- memasukkan template original atau template dengan perubahan ringan ke paket template berbayar;
- menjual versi yang hanya berisi perubahan kosmetik kecil;
- mengklaim sebagai pembuat original;
- menggunakan brand atau identitas ExcelGratis tanpa izin;
- menghapus attribution atau ownership notice yang sudah ada di file.

## Klarifikasi

Data yang Anda masukkan tetap menjadi data Anda. Lisensi ini tidak memindahkan kepemilikan copyright atas template original.

Deliverable klien yang sudah selesai atau sudah diisi data boleh diberikan kepada klien. Namun, template original atau template kosong yang dapat dipakai ulang tidak boleh diberikan kepada klien sebagai produk yang dapat dijual ulang atau didistribusikan kembali.

ExcelGratis tidak menjamin template cocok untuk setiap situasi. Pengguna tetap perlu memeriksa formula, asumsi, data, dan hasil sebelum memakai template untuk keputusan penting.
