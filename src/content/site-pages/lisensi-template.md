Halaman ini menjelaskan lisensi penggunaan website ExcelGratis, bukan nasihat hukum.

## Penggunaan yang diizinkan

Anda boleh menggunakan template untuk kebutuhan pribadi atau internal bisnis, memodifikasinya untuk kebutuhan sendiri, mengisinya dengan data Anda, membuat laporan internal, serta menggunakan hasilnya dalam pekerjaan atau layanan kepada klien selama file kosong atau original tidak dijual atau didistribusikan sebagai produk tersendiri.

## Penggunaan yang dilarang

Anda tidak boleh menjual ulang, mengunggah ulang ke website lain, menyediakan mirror unduhan, mendistribusikan sebagai koleksi sendiri, menghapus attribution atau ownership notice yang ada, mengklaim sebagai pembuat original, memasukkan file ke paket berbayar, menjual versi dengan perubahan minor saja, atau memakai brand ExcelGratis tanpa izin.

## Klarifikasi

Data yang Anda masukkan tetap milik Anda; lisensi ini tidak memindahkan copyright template. Beberapa template dapat memiliki ketentuan khusus, dan ketentuan jelas pada halaman detail template mengalahkan ketentuan umum ini. ExcelGratis tidak menjamin template cocok untuk setiap situasi.
