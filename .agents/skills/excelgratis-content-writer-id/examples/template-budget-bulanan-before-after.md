# Golden Style Example: Template Budget Bulanan

Contoh ini menunjukkan perubahan gaya, bukan replacement final seluruh halaman. Fakta harus tetap diverifikasi terhadap workbook terbaru.

## Before pattern

> Gajian sering habis sebelum akhir bulan padahal tidak merasa boros? Biasanya bukan karena boros, tapi karena tidak ada gambaran jelas ke mana uang itu pergi. Template budget bulanan ini membantu kamu melihat itu dalam hitungan menit, tanpa perlu jago Excel atau paham istilah akuntansi.

## After pattern

> Atur budget bulanan tanpa membuat rumus dari nol. Isi target anggaran dan transaksi harian, lalu dashboard menampilkan total pemasukan, pengeluaran, sisa uang, serta kategori yang sudah mendekati atau melewati batas.

## Before usage pattern

Enam langkah dengan beberapa tindakan dan penjelasan panjang pada tiap langkah.

## After usage pattern

```markdown
## Cara menggunakan template

1. **Atur target.** Buka `Anggaran Bulanan`, lalu ganti target setiap kategori.
2. **Catat transaksi.** Isi tanggal, kategori, jenis transaksi, dan jumlah di `Catatan Transaksi`.
3. **Periksa dashboard.** Buka `Dashboard Ringkasan` untuk melihat realisasi, sisa uang, dan status anggaran.
4. **Mulai bulan baru.** Simpan salinan file, lalu kosongkan transaksi lama tanpa menghapus rumus.
```

## Principle demonstrated

- Hasil muncul di awal.
- Kalimat lebih pendek.
- Nama sheet tetap spesifik.
- Repetisi benefit dihapus.
- Instruksi dipisahkan dari motivasi.
