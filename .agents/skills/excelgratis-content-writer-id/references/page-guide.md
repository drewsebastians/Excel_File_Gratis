# Guide Page Playbook

## Primary intent

`instructional`.

## Recommended structure

1. Hasil yang akan diperoleh
2. Prasyarat
3. Contoh data atau kondisi awal
4. Langkah-langkah
5. Cara memeriksa hasil
6. Error umum
7. Variasi atau langkah berikutnya

## Rules

- Beri hasil akhir di awal.
- Gunakan nama menu dan tombol Excel secara tepat.
- Satu langkah berisi satu tindakan utama.
- Jangan menggabungkan langkah dan penjelasan teori panjang.
- Tambahkan catatan hanya saat mencegah kesalahan.
- Bedakan instruksi Excel desktop, web, atau Google Sheets jika perilakunya berbeda.
- Jangan mengasumsikan versi Excel.
- Berikan checkpoint setelah langkah penting.

## Step format

```markdown
1. **Pilih tabel sumber.** Klik salah satu sel di dalam data.
2. **Buka Power Query.** Pilih `Data > From Table/Range`.
3. **Periksa header.** Pastikan setiap kolom memiliki nama unik.
```

## Verification

Selalu jelaskan bagaimana pengguna tahu langkahnya berhasil.
