# Collection Page Playbook

## Primary intent

`discovery + navigation`.

## Recommended structure

1. Untuk siapa koleksi ini
2. Masalah yang diselesaikan
3. Urutan resource yang disarankan
4. Resource unggulan dan alasan pemilihan
5. Cara memilih resource
6. Langkah berikutnya

## Rules

- Jangan membuat daftar link tanpa konteks.
- Jelaskan perbedaan tiap resource.
- Hindari deskripsi yang sama dengan card atau meta description.
- Susun urutan berdasarkan workflow pengguna.
- Jangan mempromosikan resource draft atau tidak tersedia.
- Jangan membuat klaim jumlah resource jika dinamis dan tidak diverifikasi.
