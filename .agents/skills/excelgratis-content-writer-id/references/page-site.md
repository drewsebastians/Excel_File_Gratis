# Site Page Playbook

## Scope

Untuk halaman seperti Tentang, Kontak, Trust, Privacy, Terms, Request Template, atau fixed CMS pages.

## Rules

- Utamakan kejelasan layanan dan tindakan.
- Jangan menggunakan tone tutorial pada halaman legal.
- Jangan mengubah arti legal atau privacy tanpa review.
- Jangan menambahkan janji response time tanpa komitmen operasional.
- Jangan membuat klaim jumlah pengguna, download, atau template tanpa data.
- Pertahankan route dan fixed slug.
- Request Template harus konsisten dengan kebijakan `noindex, follow` repository.
- Jelaskan penggunaan data secara spesifik pada halaman privacy.
