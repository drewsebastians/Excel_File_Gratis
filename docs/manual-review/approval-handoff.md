# Approval Handoff

Review the guide and template CSV files, then reply using only this structure. Use the exact slug from the CSV. Omit a section only when it has no entries.

```text
APPROVED_GUIDES:
- panduan-conditional-formatting-sebagai-alarm

APPROVED_TEMPLATES:
- template-checklist-acara

REJECTED_OR_NEEDS_FIX:
- template-rencana-proyek-sederhana: print preview clips the summary chart on A4 landscape
```

`APPROVED_GUIDES` grants editorial release approval only. `APPROVED_TEMPLATES` means the owner has checked the original workbook in Microsoft Excel, accepted its stated limitations, and approves release timing. Sensitive templates still require an explicit entry in `APPROVED_TEMPLATES`; no broad statement approves a group.

Do not use ambiguous responses such as “looks good”, “continue”, “the first batch is fine”, or “approve the templates”. A missing slug remains unapproved and unpublished.
